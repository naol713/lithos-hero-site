# Lithos Hero

Full-screen cinematic hero section with a cursor-following spotlight reveal, built with React 18 + TypeScript + Vite + Tailwind CSS.

## Run it

```bash
npm install
npm run dev
```

Then open the local URL Vite prints (usually `http://localhost:5173`) in your browser. Move your cursor over the hero to see the second image reveal through the soft circular mask.

## Build for production

```bash
npm run build
npm run preview
```
